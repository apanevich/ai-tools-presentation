# Презентация «ИИ-инструменты: обзор, сравнение и практика»

Данный репозиторий содержит исходный код презентации в формате **LaTeX Beamer** (тема `metropolis`).  
Презентация подготовлена для выступления на тему современного состояния экосистемы ИИ-инструментов — от чат-интерфейсов до агентов и локальных моделей, с акцентом на практическое тестирование и доступность в России.

---

## 📁 Структура

- `main.tex` — основной файл презентации.
- `images/` — папка для логотипов и иконок (должна быть создана отдельно).
- `README.md` — этот файл.

---

## 🔧 Требования для локальной сборки

Для компиляции необходима рабочая установка **TeX Live** (или **MiKTeX**) с поддержкой **LuaLaTeX** и следующих пакетов:

- `beamer`
- `metropolis` (тема)
- `booktabs`, `tabularx`, `multirow` (для таблиц)
- `graphicx` (для вставки иконок)
- `hyperref`
- `babel` с опцией `russian`
- шрифты `T2A` (кириллица)

Все перечисленные пакеты входят в стандартные дистрибутивы TeX, но могут потребовать дополнительной установки, если используется минимальная сборка.

## 🛠 Инструкция по сборке

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/your-username/ai-tools-presentation.git
cd ai-tools-presentation
```

### 2. Подготовьте изображения (опционально)

В коде используются команды для вставки логотипов (`\iconOpenAI`, `\iconGoogle` и т.д.).  
Чтобы они отображались, создайте папку `images/` и поместите в неё файлы:

- `openai.png`
- `google.png`
- `deepseek.png`
- `yandex.png`
- `anthropic.png`
- `ollama.png`

Если файлы отсутствуют, презентация всё равно скомпилируется, но иконки не появятся (будут пустые места).  
Вы можете закомментировать вызовы иконок в тексте или использовать альтернативные пути.

### 3. Скомпилируйте презентацию

```bash
lualatex main.tex
```

Для корректного отображения ссылок и оглавления может потребоваться **двукратный** запуск:

```bash
lualatex main.tex
lualatex main.tex
```

### 4. Результат

После успешной компиляции в текущей папке появится файл `main.pdf` – готовая презентация.

## 🌐 Альтернативный способ – Overleaf

Если у вас нет локальной установки TeX, можно использовать [Overleaf](https://www.overleaf.com/):

1. Загрузите файл `main.tex` в новый проект.
2. Убедитесь, что в настройках компилятора выбран **LuaLaTeX** (Menu → Compiler → LuaLaTeX).
3. Нажмите «Recompile».
4. Презентация будет собрана онлайн без установки дополнительного ПО.
